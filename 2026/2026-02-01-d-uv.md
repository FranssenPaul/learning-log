# Learning Log — UV

## What UV is

**UV** is a fast Python package and project manager.

It replaces several traditional tools:

- `pip`
- `venv`
- parts of Poetry or Pipenv

---

## Why it helps

Instead of manually creating an environment, activating it, installing packages, and then running a command, UV can do the whole flow for you.

```bash
uv run uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

---

## Common commands

```bash
uv venv
uv pip install fastapi uvicorn[standard]
uv run python script.py
uv run uvicorn main:app
```

---

## Typical setup

```toml
[project]
name = "backend"
requires-python = ">=3.10"
dependencies = [
    "fastapi>=0.128.0",
    "uvicorn[standard]>=0.40.0",
]
```

With that file in place, `uv run ...` can create the environment, install dependencies, and run the command.
