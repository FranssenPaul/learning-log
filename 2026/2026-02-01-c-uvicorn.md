# Learning Log — Uvicorn

## What Uvicorn is

**Uvicorn** is the ASGI server that runs a FastAPI app and listens for network requests.

Without it, FastAPI code is just Python code sitting in a file.

---

## What Uvicorn does

- opens a port
- listens for TCP connections
- parses HTTP requests
- forwards them to the ASGI app
- sends the response back to the client

```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

---

## Relationship with FastAPI

Simple picture:

```text
Browser <-> Uvicorn <-> FastAPI
```

- Uvicorn speaks HTTP to the browser
- FastAPI handles application logic
- Uvicorn turns FastAPI responses into real network responses

---

## Command breakdown

- `main:app`: import `app` from `main.py`
- `--reload`: restart on file changes
- `--host 0.0.0.0`: listen on all interfaces
- `--port 8000`: use port 8000
