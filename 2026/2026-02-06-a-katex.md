## KaTeX `\phantom` Deep Dive

`\phantom` is the key trick for empty blanks that still reserve visual space.

### What `\phantom` does

- `\phantom{...}` keeps width + height + depth of content, but content is invisible.
- `\hphantom{...}` keeps horizontal width only.
- `\vphantom{...}` keeps vertical height/depth only.

In our current blank system:

- `underline`: `\underline{\phantom{0}}`
- `box`: `\boxed{\phantom{0}}`
- `tinted`: `\colorbox{#1f2937}{\phantom{0}}`

So the blank has the dimensions of `0`, but no visible character.

### Why `\phantom{0}` is a good default

- Stable baseline in math mode.
- Good fit for digit answers.
- Predictable shape inside underline/box/colorbox.

To make blanks wider:

- one char: `\phantom{0}`
- two chars: `\phantom{00}`
- expression-sized blank: `\phantom{x^2}`

### Reveal behavior in our renderer

The same container style is kept, answer is inserted inside:

- underline reveal: `\underline{\color{#4ade80}{2}}`
- boxed reveal: `\boxed{\color{#4ade80}{2}}`
- tinted reveal: `\colorbox{#1f2937}{\color{#4ade80}{2}}`

### Practical patterns

Single blank:

```tex
\underline{\phantom{0}}
```

Two-character blank:

```tex
\boxed{\phantom{00}}
```

Tinted expression-sized blank:

```tex
\colorbox{#1f2937}{\phantom{x^2}}
```

`hphantom` only (horizontal reserve):

```tex
\underline{\hphantom{000}}
```

`vphantom` only (vertical alignment):

```tex
A_{\vphantom{\frac{1}{2}}}
```

### Rule of thumb

- Use `\phantom{0}` for digit answers.
- Use `\phantom{x}` for letter-heavy answers.
- Use repeated blanks (`<<__>>`) when you want one slot per character.
- Use a wider phantom (`\phantom{00}` or `\phantom{x^2}`) when one blank should accept multiple characters.

### To Write in a phantom
Rerender with the answer inside the phantom:

```tex
\colorbox{#1f2937}{\color{#4ade80}{2}}
```
