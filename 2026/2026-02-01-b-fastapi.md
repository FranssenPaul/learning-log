# Learning Log — FastAPI Basics

## What FastAPI is

**FastAPI** is a modern Python framework for building APIs.

You define routes, receive requests, validate data, and return responses.

---

## Mental model

Think of FastAPI as the part of your application that knows:

- which endpoints exist
- which parameters they accept
- what shape the response should have

```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/health")
def health():
    return {"ok": True}
```

---

## What it does well

- route declaration
- request parsing
- automatic JSON responses
- type-based validation
- automatic docs at `/docs`

---

## Key idea

FastAPI defines the API logic, but it still needs a server to run it.
