---

## How it works

- `i18n.js` initializes `i18next`
- React is connected through `initReactI18next`
- the language is loaded from `localStorage`
- `document.lang` and `document.dir` are updated on language change

---

## Persistence

The chosen language is stored in `localStorage` with `pc_language`.

That gives:

- persistence after reload
- accessibility and SEO improvements through `lang`
- RTL support through `dir`

---

## Example

```jsx
import { useTranslation } from 'react-i18next'

function Title() {
  const { t, i18n } = useTranslation()

  return (
    <>
      <h1>{t('vectors.title')}</h1>
      <select value={i18n.language} onChange={(e) => i18n.changeLanguage(e.target.value)}>
        <option value="fr">Français</option>
        <option value="en">English</option>
      </select>
    </>
  )
}
```
